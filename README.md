## Project Overview

This project demonstrates core Java concepts through step-by-step files, each focusing on a specific topic. The structure is organized under the `com.piyush.net` package.

### Topics Covered

- **Data Types**  
  See [src/com/piyush/net/Step01Datatypes.java](src/com/piyush/net/Step01Datatypes.java): Primitive and non-primitive types, arrays, lists.

- **Variable Scopes**  
  See [src/com/piyush/net/Step02VariableScopes.java](src/com/piyush/net/Step02VariableScopes.java): Static, instance, and local variables.

- **Loops**  
  See [src/com/piyush/net/Step03Loops.java](src/com/piyush/net/Step03Loops.java): For, for-each, while, do-while, break, continue, forEach.

- **Exception Handling**  
  See [src/com/piyush/net/Step04ExceptionHandling.java](src/com/piyush/net/Step04ExceptionHandling.java): Try-catch-finally, common exceptions.

- **Conditionals**  
  See [src/com/piyush/net/Step05Conditionals.java](src/com/piyush/net/Step05Conditionals.java): If-else, else-if, ternary operator.

- **Inbuilt Methods**  
  See [src/com/piyush/net/Step06InbuiltMethods.java](src/com/piyush/net/Step06InbuiltMethods.java): String, Math, System methods.

- **Classes & Objects**  
  See [src/com/piyush/net/Step07Class.java](src/com/piyush/net/Step07Class.java): Class definition, static block.

- **Methods & Constructors**  
  See [src/com/piyush/net/Step08Methods.java](src/com/piyush/net/Step08Methods.java) and [src/com/piyush/net/Step08MethodsConstructors.java](src/com/piyush/net/Step08MethodsConstructors.java): Static methods, setters, constructor types.

- **Structural Elements**  
  See [src/com/piyush/net/Step09StructuralElements.java](src/com/piyush/net/Step09StructuralElements.java): Classes vs objects, constructors, access modifiers, keywords.

- **OOP Pillars**  
  - Abstraction: [src/com/piyush/net/Step10OopsPillars1Abstraction_Interface.java](src/com/piyush/net/Step10OopsPillars1Abstraction_Interface.java), [src/com/piyush/net/Step10OopsPillars1Abstraction_AbstractClassAndMethod.java](src/com/piyush/net/Step10OopsPillars1Abstraction_AbstractClassAndMethod.java)
  - Encapsulation: [src/com/piyush/net/Step10OopsPillars2Encapsulation.java](src/com/piyush/net/Step10OopsPillars2Encapsulation.java)
  - Inheritance: [src/com/piyush/net/Step10OopsPillars3Inheritance.java](src/com/piyush/net/Step10OopsPillars3Inheritance.java)
  - Polymorphism: [src/com/piyush/net/Step10OopsPillars4Polymorphism.java](src/com/piyush/net/Step10OopsPillars4Polymorphism.java), [src/com/piyush/net/Step10OopsPillars4PolymorphismProduction.java](src/com/piyush/net/Step10OopsPillars4PolymorphismProduction.java)
  - Summary: [src/com/piyush/net/Step10OopsPillars.java](src/com/piyush/net/Step10OopsPillars.java)

- **Project Structure & Packages**  
  See [src/com/piyush/net/Step12Package_ProjectStructure.java](src/com/piyush/net/Step12Package_ProjectStructure.java): How packages organize code.

- **ENUMs & Profiles**  
  See [src/com/piyush/net/Step13ENUM.java](src/com/piyush/net/Step13ENUM.java): Enum usage.  
  See [src/com/piyush/net/Step14SprintProfiles.java](src/com/piyush/net/Step14SprintProfiles.java): Spring profiles for environment-specific beans.

- **Working with Files & APIs**  
  See [src/com/piyush/net/Step15WorkingWithFiles.java](src/com/piyush/net/Step15WorkingWithFiles.java): File operations.  
  See [src/com/piyush/net/Step16APIs.java](src/com/piyush/net/Step16APIs.java): API basics.

### Entry Point

- The main method is in [src/com/piyush/net/main.java](src/com/piyush/net/main.java):  
  Run with:
  ```sh
  javac com/piyush/net/main.java
  java com.piyush.net.main
  ```

### How to Use

- Each step file contains code and comments explaining the concept.
- Uncomment relevant lines in [src/com/piyush/net/main.java](src/com/piyush/net/main.java) to run specific examples.
